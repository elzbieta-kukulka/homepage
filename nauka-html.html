Czego się uczę

HTML

ang. HyperText Markup Language - język znaczników. Odpowiada za treść ( a właściwie za jej właściwe opakowanie). Treść czyli to wszystko co jest nośnikiem informacji. CONTENT IS THE KING

HTML to szkielet strony internetowej. Jedna głowa - head i jedno ciało - body.
main - główna część dokumentu

Tag- znaczniki

section - sekcja dokumentu - każda sekcja powinna mieć nagłówek
article - samodzielne treści
header - nagłówek dokumentu/sekcji/artykułu
footer - stopka
h1-h6 - tekstowe nagłówki
p - paragraf

br lub br/ - nowa linia

b i i - nie mają dużego znaczenia semantycznego - służą do wyróznienia teksstu ztekstu

strong podkreśla wysoką istotność i poważność frazy w nim zawartej a em

podkreśla, że tekst może mieć inne znaczenie np. ironia

Linki łącza wewnątrz dokumentu

a href="nazwa" target="">opis łącza /a

Nawigacja w obrębie dokumentu

element id="mój-identyfikator" - atrybut id identyfikuje element (przydaje się do nawigacji w obrębie dokumentu) - np. section id="notatki" --> a href="#notatki>... koniec a

NAV - powtarzalna część nawigacji

Visual Studio Code

CTRL+SHIFT+P - wpisać format - wybrać format dokument

Nazywanie rzeczy

  • nazwy zapisujemy bez polskich znaków
  • nie używamy spacji
  • najlepiej używać angielskich słów
  • trzymamy się jednego schematu

Konwencja nazewnictwa

  • camelCase - mySuperBoldHeader
  • kebab-case - my-super-bold-header - używany do atrybutu id
  • snake_case - my_super_bold_header
  • PascalCase - MySuperBoldHeader
