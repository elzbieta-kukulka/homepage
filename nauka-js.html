JavaScript

ECMAScript 2015+(ES6+)

Jest to ustandaryzowany przez organizację ECMA skryptowany język programowania

JavaScript implementuje rzeczy ze specyfikacji ECMAScript

JS składa się z:

  • wartości - dla JS to to, czym liczby dla matematyki
  • literały
  • operacje porównywania
  • zmienne
  • funkcje
  • obiekty

Pliki zawierające kod JS mają rozszerzenie .js. Podobnie jak style mają też swój znacznik - script, dodajemy go na samym dole HTML tuż przed zamknięciem znacznika body - script src="...">zamnknięcie script

Zmienna

const name = "ela'; - stworzenie powiązania między nazwą a zmienną

const age = 32;

Backtick/backquote - ` - używamy, gdy chcemy użyć literału np. console.log(`Nazywam się ${name} i mam ${age} lat`)

Typy danych i immutability

Zmienna to takie połączenie z naszym wszechświatem wartości. Zmienna to nie wartość!

  • const - zmienne mające stałe połączenie z jedną wartością
  • let - zmienne, których połączenie możemy przepinać, ale definiujemy tylko raz, np. let isHappy = false; isHappy = true;

Do zmiennej można przepisać wszystko kiedy ją definiujemy. Zmienne w JS są jak młody pelikan: "łykną" wszystko.

Typy proste w JS (primitives)

  • String - tekst, otaczany ' lub "
  • Number - liczba
  • Bigint - duża liczba, większa niż 2 do potęgi 53
  • boolean - true lub false
  • null - wartość celowo niezdefiniowana
  • undefined - brak definicji
  • symbol - tworzy "unikalne opakowanie"

IMMUTABILITY (niezmienność, niemutowalność)

Nie możemy zmieniać wartości primitive value w JS. To co zmieniamy to połączenie między nimi, a zmiennymi.
