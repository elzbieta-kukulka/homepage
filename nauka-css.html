CSS

Co to jest, składnia i dodawanie stylów do dokumentu

CSS = Cascading Style Sheet - Kaskadowe arkusze stylów - odpowiadają za prezentację, wpływa na wygląd elementów HTML na ekranie, ale też wydrukach

Składnia CSS to reguły: selektor { właściwość: wartość; }

Dodanie stylów do html:

  • w head tag style i między nim reguły
  • w head link href="adres" rel="stylesheet" type="text/css"

Prawdziwe selektory

1. .(kropka) to selektor klasy - znacznik class="name">treść /> - odwołanie to .name {własćiwość: wartość}

2. #(hash) - selektor id - znacznik id="name"> treść /> - jedno id na stronie

styl liniowy style=""

element

Jeśli specyficzność jest równa - pierwszeństwo ma selektor "bliżej" elementu. Wniosek: dodawać klasy do elementów i unikać zagnieżdżania.

Nazewnictwo klas:

  • używaj pojedyńczych słów
  • nazywaj rzeczy zgodnie z ich funkcją na stronie
  • startuj z małej litery np. header, a nie Header
  • nie zaczynaj od cyfry
  • jeśli musisz użyć dwóch słów, użyj tzw. kebab-case np news-listing
  • staraj się, żeby element miał jedną klasę (nie zwiększaj specyficznośći bez potrzeby

Display, wymiary, overflow

div - nie ma znaczenia semantycznego, używamy gdy chcemy "opakować" (tylko, kiedy nie istnieje odpowiedni semantyczny znacznik)

Właściwość display:

  • display: block - elementy ustawiają się jedno pod drugim
  • display: inline - wyświetlany liniowo, zajmuje tyle miejsca, ile zajmuje content
  • display: inline-block - zachowuje się jak element liniowy, ale dostaje właściwości elementu blokowego, bo normalnie element liniowy ich nie ma. Czyli:
    • width - szerokość
    • height - wysokość
    • max-width/max-height - domyślnie na 100%, częśćiej używany max-width
    • min-width/min-height

Właściwość overflow:

  • overflow: hidden - ucina, chowa wszytsko co wystaje
  • overflow: visible - domyślna wartość
  • overflow: auto - dokłada scroll w elemencie

Jednostki:

  • 0 - zapisujemy bez jednostek
  • px - podstawowa jednostka np 600px
  • % - % szerokości lub wysokości elementu rodzica
  • vw - viewport width - szerokość okna
  • viewport height - wysokość oknanp 100vh - cała wysokość okna

BOX MODEL

PADDING

wewnętrzny margines - odpowiadaa za to, o ile musi być odsunięty element od elementu, który ma padding

Zapis: padding-top: 2px; padding-right: 2px; padding-bottom: 2px; padding-left: 2px; odpowiada padding: 2px 2px 2px 2px; lub wartości "lustrzane" - padding: 2px 2px; lub jeśli wszystkie wartości są takie same to padding: 2px;

BORDER

to też część elementu. Zapis pełny/zapis skrótwoy jak przy paddingu:

border-width: 1px 2px 3px 4px lub 10px lub 10px 20px

border-style: solid lub dotted

border-color: #fff

lub border: 2px solid #fff

lub border-top: 3px solid hotpink; i każde po kolei jeśli mają być inne ramki.

MARGIN

margines - nie jest częścią elementu, działa przy właściwościach blokowych, odpowiada za to, o ile możemy się odsunąć od innych elementów

  • zapisujemy tak samo jak padding,
  • marginesy się sumują
  • własność auto - margin: 20px auto - elementy się wyśrodkowują, przy domyślnym display: block, zadziała tylko jeśli szerokość jest zdefiniowana
